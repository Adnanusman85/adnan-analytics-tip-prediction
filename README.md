# Tip Prediction Using Multiple Linear Regression
### By Adnan Analytics — Turning data into clarity.

This project analyses a real-world restaurant dataset and builds a predictive model capable of estimating tip amounts using multiple linear regression in Excel.

## 📊 Project Summary
- 244 observations
- Variables: sex, smoker status, day, time, party size, total bill, tip
- Model: Multiple Linear Regression (Excel Data Analysis Toolpak)
- R²: 0.4700
- Adjusted R²: 0.4392
- RMSE: 1.0339
- Average percentage error: 34%

## 🔍 Key Insight
The total bill amount is the strongest predictor of tip size.
